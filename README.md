## How to start

 - install xampp
 - install mysql if you need
 - install navicat
 - install php
 - install composer
 - copy .env.local .env
 - run install.bat commands
 - run update.bat

### mail configuration
 - make sure you do configuration for email verification in env file
```  MAIL_MAILER=smtp
  MAIL_HOST=mailpit
  MAIL_PORT=1025
  MAIL_USERNAME=null
  MAIL_PASSWORD=null
  MAIL_ENCRYPTION=null
  MAIL_FROM_ADDRESS="hello@example.com"
  MAIL_FROM_NAME="${APP_NAME}"
  ```
